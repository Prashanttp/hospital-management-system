CareSync Pro

Enterprise Spring Boot & Data JPA Hospital Management System

🚀 Project Overview

CareSync Pro is a full-stack, enterprise-grade healthcare operations platform built as a Final Year Computer Science Capstone Project. It provides secure role-based access control (RBAC), patient health record management, clinical doctor consultations with digital prescriptions, and real-time hospital operations analytics.

🛠️ Technology Stack

  • Java 21
  • Spring Boot 3.5.3
  • Spring Data JPA / Hibernate
  • Spring Security 6 (Stateless JWT)
  • PostgreSQL Managed Database
  • Lombok & ModelMapper
  • Glassmorphic Web Frontend
  • Docker & Render Cloud Deployment
